
# Enterprise Revenue Leakage & Customer Lifetime Value Analytics Platform

## Business Context
Leadership observed revenue decline without proportional churn increase.
This platform identifies hidden revenue leakage drivers and quantifies CLV impact.

## KPIs
- Revenue Leakage %
- Revenue at Risk
- CLV Loss
- Retention Uplift Opportunity

## Architecture
- Data Validation & Feature Engineering
- Segmentation & Statistical Analysis
- ML-based Revenue-Risk Identification
- Executive Streamlit Dashboard

## How to Run
1. python -m venv venv
2. venv\Scripts\activate
3. pip install -r requirements.txt
4. streamlit run src/visualization/dashboard.py

## Interview Defense
This system mirrors an internal MNC analytics platform focused on financial impact,
not churn-only modeling.
